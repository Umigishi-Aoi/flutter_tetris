This repository is made for the game like tetris by Flutter

Game link : https://umigishi-aoi.github.io/flutter_tetris/

You can play this game by button and keyboad.

Key config:

- enter:  Start
- arrowRight:  Right
- arrowLeft:  Left
- arrowDown:  Down
- keyX:  R90
- keyZ:  L90
- keyC:  Keep
- arrowUp:  HardDrop
